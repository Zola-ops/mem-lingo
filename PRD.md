# Mem-Lexis 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称与定位

**产品名称**: Mem-Lexis (记忆语言学习)

**一句话定位**: 融合电子宠物养成机制的英语学习平台，让学习充满成就感和乐趣。

### 1.2 目标用户

- 英语学习者（大学生、考研党、留学备考群体）
- 希望通过游戏化方式提升学习动力的人群
- 喜欢收集和养成元素的用户

### 1.3 核心价值主张

> "学习不再是枯燥的任务，而是充满成就感的养成体验"

## 2. 用户旅程与核心场景

### 2.1 新用户首次使用流程

```
打开页面 → 欢迎引导 → 选择目标考试 → AI能力测评 → 孵化宠物 → 开始学习
```

1. **欢迎页**: 展示产品Logo和slogan，提供"开始测评"按钮
2. **考试选择**: 四级/六级/考研/托福/雅思，选择目标考试
3. **能力测评**: 10道词汇选择题，AI生成能力雷达图
4. **宠物孵化**: 根据测评结果随机生成宠物（类型+颜色），显示宠物形象和名字
5. **完成引导**: 进入学习中心主页

### 2.2 日常学习流程

```
打开App → 查看宠物状态 → 学习新单词/复习 → 打卡 → 喂养宠物
```

1. **首页**: 显示宠物状态（饥饿值/开心值）、今日任务进度
2. **词汇学习**: 闪卡模式展示单词，可标记已学
3. **复习**: 基于艾宾浩斯遗忘曲线安排待复习单词
4. **打卡**: 完成每日任务后打卡获得积分和连击奖励
5. **宠物互动**: 消耗积分喂食、遛宠物、抚摸

### 2.3 宠物死亡与重置

```
宠物饱食度归零 → 死亡动画 → 清除学习记录 → 重新孵化宠物 → 保留用户设置
```

## 6. 艾宾浩斯复习算法

### 6.1 复习间隔规则

| 次数 | 间隔 |
|------|------|
| 1 | 20分钟 |
| 2 | 1小时 |
| 3 | 9小时 |
| 4 | 1天 |
| 5 | 2天 |
| 6 | 6天 |
| 7+ | 上次间隔 × 容易度 |

### 6.2 难度评分

- 完全忘记 (0-2): 间隔重置为20分钟，容易度-0.2
- 有点模糊 (3-4): 间隔×0.8，容易度不变
- 记得 (5-6): 间隔×1.2，容易度+0.1
- 完美记住 (7-8): 间隔×1.5，容易度+0.2

### 6.3 下次复习时间计算

```
下次复习时间 = 现在时间 + (基础间隔 × 容易度因子)
```

## 7. 宠物系统规则

### 7.1 宠物属性

| 属性 | 范围 | 衰减规则 |
|------|------|----------|
| 饥饿值 | 0-100 | 每日-10 |
| 开心值 | 0-100 | 每日-5 |
| 经验值 | 0-升级 | 学习/互动获得 |

### 7.2 升级规则

```
升级所需经验 = 等级 × 100
```

### 7.3 宠物死亡条件

- 饥饿值 ≤ 0 时宠物死亡
- 死亡后清除: learnedWords, wordMastery, reviewRecords, checkin.streak
- 保留: selectedExam, petName, 积分余额

### 7.4 喂养消耗

| 食物 | 价格 | 恢复饥饿 |
|------|------|----------|
| 苹果 | 5积分 | +20 |
| 披萨 | 10积分 | +40 |
| 蛋糕 | 20积分 | +70 |
| 神秘大餐 | 50积分 | +100 |

### 7.5 互动消耗

| 动作 | 消耗 | 效果 |
|------|------|------|
| 抚摸 | 3积分 | 开心+5 |
| 遛宠物 | 10积分 | 随机属性+5~15 |
| 一同学习 | 0积分 | 经验+10（每日限3次） |
