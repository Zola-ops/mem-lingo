# Mem-Lexis 英语学习 + 电子宠物网页版 v2

## 概念与愿景

**Mem-Lexis** - 一个融合了电子宠物养成机制的学习平台。用户通过每日学习打卡来喂养和培养自己的小宠物，宠物的成长状态与学习进度绑定。学习不再是枯燥的任务，而是充满成就感的养成体验。

**核心特色**: AI 能力测评 → 生成宠物 + 雷达图 → 宠物养成与学习联动

- **随机顺序 (random)**: 每次展示随机打乱

## 功能与交互

### 1. 首次引导流程

#### 1.1 欢迎页
- Logo + 标题展示
- "开始测评" 按钮

#### 1.2 考试选择
- 展示所有考试类型: CET-4, CET-6, 考研, 托福, 雅思
- 用户必须选择一个作为目标
- 可跳过（默认 CET-4），但之后可切换

#### 1.3 能力测评
- 词汇量测试: 10道选择题，测试词汇量等级
- 听力测试: 听句子选择意思
- 阅读测试: 读短文回答问题
- AI 根据测试结果生成能力雷达图

#### 1.4 宠物孵化
- 根据能力测评结果，随机生成宠物外观（不同颜色/形状）
- 宠物属性与雷达图能力映射
- 显示宠物名字（可自定义）

### 2. 学习中心

#### 2.1 首页设计
- 漂浮单词展示: 随机词汇显示在屏幕中上部，带有浮动动画效果
- 四个功能板块: 词汇学习、复习、词汇测验、收藏
- "复习"按钮在无学习记录时显示为灰色禁用状态
- 学习概览统计: 已学习、已掌握、收藏、待复习
- 艾宾浩斯遗忘曲线说明

#### 2.2 考试选择器
- 支持考试: CET-4, CET-6, 考研, 托福, 雅思
- 切换考试后，学习内容实时筛选
- 当前选中考试高亮显示

#### 2.3 艾宾浩斯复习系统
- 基于遗忘曲线理论安排复习时间
- 复习间隔: 20分钟 → 1小时 → 9小时 → 1天 → 2天 → 6天
- 每次学习/复习后自动计算下次最佳复习时间
- 记录项包含: 下次复习时间、间隔天数、容易度因子、重复次数
- 复习时根据记忆难度评分(完全忘记/有点模糊/记得/完美记住)

#### 2.4 词汇学习
- 核心词汇区（标记为"核心"的词）
- 全部词汇列表
- 单词卡: 单词 + 音标 + 释义 + 例句
- 点击发音按钮: 使用 Web Speech API (英式英语)
- AI 问答: 选词进入 AI 对话模式
- 标记已学后自动创建艾宾浩斯复习记录

#### 2.4 听力训练
- 听力材料列表（按难度分类）
- 点击播放: 使用 HTML5 Audio
- 完成练习获得积分

#### 2.5 口语练习
- 跟读练习界面
- 使用 Speech Recognition API 进行语音识别
- AI 发音评分

#### 2.6 阅读理解
- 文章列表（按难度分类）
- 文章详情: 原文 + 题目 + 答案解析
- AI 问答: 针对文章内容提问

#### 2.7 写作
- 写作模板列表
- 模板详情: 结构 + 常用表达
- AI 批改: 提交作文获得 AI 评分和建议

### 3. 宠物系统

#### 3.1 能力雷达图
- 5维度: 词汇、听力、口语、阅读、写作
- 每维度 0-100 分
- 雷达图颜色与宠物颜色呼应
- 能力数值直接影响宠物属性

#### 3.2 宠物展示区
- 宠物形象（随机生成的可爱角色）
- 宠物名称（可自定义）
- 宠物状态: 饥饿值 / 开心值 / 经验值 / 寿命
- 等级系统: Lv.1 - Lv.99

#### 3.3 宠物动画
- idle: 站立/眨眼
- happy: 跳跃/旋转
- hungry: 低头/摇晃
- eating: 吃东西
- sleeping: 睡觉
- sick: 生病（饥饿值过低）
- dead: 死亡动画

#### 3.4 宠物死亡机制
- 饱食度 ≤ 0 时宠物死亡
- 死亡后: 清除所有学习记录、重置宠物、保留用户选择
- 弹出警告提示

#### 3.5 喂养机制
- 学习积分兑换食物
- 积分通过完成任务获得
- 食物种类: 苹果(5券)、披萨(10券)、蛋糕(20券)、神秘大餐(50券)

#### 3.6 宠物互动
- 抚摸: 提升开心值（消耗积分）
- 遛宠物: 消耗积分，提升随机属性
- 和宠物一起学习: 学习效率提升

### 4. 打卡系统

#### 4.1 今日任务
- 每日学习目标:
  - 学习10个单词
  - 完成1篇阅读
  - 口语练习1次
- 进度条展示

#### 4.2 打卡动作
- 任务完成后才能打卡
- 打卡成功: 播放庆祝动画 + 宠物反应 + 获得积分
- 连击奖励: 连续打卡天数越多，奖励越丰厚

#### 4.3 打卡记录
- 日历视图展示打卡历史
- 连击天数显示

### 5. AI 功能

#### 5.1 AI 能力测评
- 根据测试结果生成雷达图
- 文字描述各维度能力等级

#### 5.2 AI 问答
- 词汇: 单词用法、例句、近义词
- 阅读: 文章理解、细节提问
- 口语: 发音评分、改进建议

#### 5.3 AI 写作批改
- 提交作文
- AI 从语法、词汇、结构、内容四方面评分
- 给出具体改进建议

### 6. 用户数据 (localStorage)

#### 6.1 存储内容
- 用户信息: 昵称、宠物名称、已选考试
- 能力数据: 雷达图各维度分数、词汇量测试结果
- 学习进度: 已学单词、已完成任务、总学习时长
- 宠物状态: 种类、等级、属性值、饱食度、开心值
- 打卡记录: 日期数组、连续天数、最长连击
- 积分余额
- 学习记录历史

## 技术方案

### 技术栈
- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，单文件运行

### 外部资源
- Google Fonts: Noto Sans SC, Nunito
- 图标: 内联 SVG 或 Unicode emoji

### 数据持久化
- localStorage 存储所有用户数据
- 每日自动衰减宠物饱食度（-10/天）

### API 使用
- Web Speech API: 语音合成 + 语音识别
- HTML5 Audio: 音频播放
- AI 功能: 模拟 AI 响应（根据关键词生成建议）
